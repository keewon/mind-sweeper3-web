# Mindsweeper3 (Minesweeper in the dark)
기존 지뢰찾기와 동일하지만, 시야가 좁아지는 지뢰찾기이다.
모바일 웹앱으로 구현한다.

첫 화면에서는 다음 메뉴를 보여준다.
- 일반 지뢰찾기 (9x9, 지뢰 10개)
- 주변 3x3 만 보이는 지뢰찾기
- 방금 선택한 곳만 보이는 지뢰찾기

따라서 기존 지뢰찾기와는 인터페이스가 조금 다르다.
- 칸 선택 시 어떤 칸인지 보여줌.
- 아래쪽에 Open/Mark 버튼이 있음
- 기타 인터페이스는 클래식 지뢰찾기와 유사하게 만든다.

고려할 점
- 시각장애인도 플레이할 수 있도록 웹접근성을 잘 지켜야 함.
